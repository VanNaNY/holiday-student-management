# holiday-student-management

假期学生管理系统 (Holiday Student Management System)

## 项目结构

本仓库采用前后端分离的单仓库（monorepo）结构：

```
holiday-student-management/
├─ frontend/   # 前端
├─ backend/    # 后端
└─ README.md   # 项目总说明（本文件）
```

- 前端代码请放在 [`frontend/`](frontend/) 目录下。
- 后端代码请放在 [`backend/`](backend/) 目录下。

## 协作约定

- 主分支为 `main`，建议各自开功能分支开发，通过 Pull Request 合并。
- 技术栈与功能范围待定，确定后会在此处补充说明。
